# node-project-default

Template Node.js + TypeScript project with:

- Express minimal app
- Winston logger
- ESLint + Prettier
- Jest + ts-jest
- Dockerfile

Quick start

1. Instale dependências:

```bash
npm install
```

2. Rodar em desenvolvimento:

```bash
npm run dev
```

3. Build e start:

```bash
npm run build
npm start
```

4. Rodar testes:

```bash
npm test
```

Arquivos importantes:

- `src/` - código fonte
- `tests/` - testes
- `Dockerfile` - imagem de produção
